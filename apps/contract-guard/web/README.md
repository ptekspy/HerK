# ContractGuard Web (Next.js)

Product UI for onboarding and org-scoped operations:

- Dashboard
- Repositories
- Services
- Policies
- Checks + check detail
- Waivers
- Notifications
- Team
- Billing

## Run

```bash
pnpm --filter web dev
```

Default port: `4000`

Set `NEXT_PUBLIC_API_URL` to point at the API (default `http://localhost:4001`).
